# 🚀 Mi Primera PWA - Nestor Studio

¡Bienvenido! Este es mi primer proyecto de **Progressive Web App (PWA)** desarrollado con HTML puro, CSS avanzado y JavaScript. El objetivo es crear una experiencia de usuario fluida, instalable y con alto impacto visual.

## ⚡️ Características principales

- **PWA Ready:** Instalable en iOS, Android y Desktop.
- **Videos High-End:** Integración de fondos animados (renders 3D y ASCII) gestionados con **Git LFS**.
- **Diseño Glassmorphism:** Interfaz moderna con efectos de desenfoque y transparencias.
- **Despliegue Continuo:** Alojado en **Vercel** con sincronización automática desde GitHub.

## 🛠️ Tecnologías utilizadas

- **Frontend:** HTML5, CSS3 (Custom Variables & Backdrop Filters).
- **Backend/Hosting:** Vercel.
- **Versionamiento:** Git + Git LFS (Large File Storage).
